# Ksheera-Sagara (Agriculture)

Ksheera-Sagara is an Android application developed to support dairy farm and agriculture management through digital solutions. The application helps farmers manage milk records, cattle information, expenses, and daily activities in a simple and organized way.

## Features
- Milk Entry Management
- Cow Management
- Expense Tracking
- Daily & Monthly Passbook
- Calendar-Based Records
- AI Suggestions
- User Login & Registration

## Technologies Used
- Kotlin
- Jetpack Compose
- Android Studio
- Firebase
- MVVM Architecture

## Objective
The main objective of this project is to simplify dairy and agriculture management using a mobile application that improves accessibility, organization, and digital record maintenance for farmers.
